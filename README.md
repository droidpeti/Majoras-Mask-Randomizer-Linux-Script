# Majora's Mask Randomizer Setup for Linux
A simple script to install ZoeyZolotova's MM Randomizer on Linux using Wine.

## Features
* Automatically installs the required .NET 5 Desktop Runtime.
* Sets up a clean Wine prefix to avoid conflicts.
* Creates a `launch_mmr.sh` shortcut for easy play.

## How to Use
1. Download the script:
   ```
   wget https://raw.githubusercontent.com/droidpeti/Majoras-Mask-Randomizer-Linux-Script/main/mm_setup.sh
   ```

2. Make it executable:
   ```
   chmod +x mm_setup.sh
   ```
3. Run it:
   ```
   ./mm_setup.sh
   ```
